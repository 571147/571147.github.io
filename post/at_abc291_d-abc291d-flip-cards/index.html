AT_abc291_d [ABC291D] Flip Cards

2023-04-25 3 min read # AtCoder # 题解

题目

  • 有 nnn 个卡片排成一排，编号为 1,2,...,n1,2,...,n1,2,...,n。
  • 第 iii 个卡片的正面是 aia_iai​，反面是 bib_ibi​。
  • 现在你可以将每个卡片翻到正面或反面，使相邻两个卡片上的数字互不相同。
  • 求方案数对 998244353998244353998244353 取模。
  • 1≤n≤2×1051\le n\le2\times10^51≤n≤2×105，1≤ai,bi≤1091\le a_i,b_i\le 10^91≤ai​,bi​≤109。

DP 做法

这道题用暴力 DFS 肯定要以 O(2n)O(2^n)O(2n) 的复杂度超时，所以考虑 DP。

定义 dpi,0dp_{i,0}dpi,0​ 表示当第 iii 个卡片正面朝上时前 iii 个卡片的放置方案数，dpi,1dp_{i,1}dpi,1​ 表示当第 iii 个卡片反面朝上时前 iii 个卡片的放置方案数，不难得出以下两个状态转移方程 (2≤i≤n)(2\le i\le n)(2≤i≤n)：

dpi,0=dpi−1,0(ai−1≠ai)+dpi−1,1(bi−1≠ai)dp_{i,0}=dp_{i-1,0}(a_{i-1}\ne a_i)+dp_{i-1,1}(b_{i-1}\ne a_i) dpi,0​=dpi−1,0​(ai−1​​=ai​)+dpi−1,1​(bi−1​​=ai​)

dpi,1=dpi−1,1(bi−1≠bi)+dpi−1,0(ai−1≠bi)dp_{i,1}=dp_{i-1,1}(b_{i-1}\ne b_i)+dp_{i-1,0}(a_{i-1}\ne b_i) dpi,1​=dpi−1,1​(bi−1​​=bi​)+dpi−1,0​(ai−1​​=bi​)

当 i=1i=1i=1 时，正面反面方案数都为 111，即 dp1,0=1,dp1,1=1dp_{1,0}=1,dp_{1,1}=1dp1,0​=1,dp1,1​=1。

最后输出 dpn,0+dpn,1dp_{n,0}+dp_{n,1}dpn,0​+dpn,1​，复杂度为 O(n)O(n)O(n)。

代码实现

#include<bits/stdc++.h>
using namespace std;
const int mod=998244353;
int n,a[200010],b[200010],dp[200010][2];
int main(){
	scanf("%d",&n);
	for(int i=1;i<=n;i++) scanf("%d%d",&a[i],&b[i]);
	dp[1][0]=1,dp[1][1]=1;
	for(int i=2;i<=n;i++){
		dp[i][0]=((a[i-1]!=a[i])*dp[i-1][0]+(b[i-1]!=a[i])*dp[i-1][1])%mod;
		dp[i][1]=((b[i-1]!=b[i])*dp[i-1][1]+(a[i-1]!=b[i])*dp[i-1][0])%mod;
	}
	printf("%d",(dp[n][0]+dp[n][1])%mod);
	return 0;
}
