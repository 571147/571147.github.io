AT_abc292_c [ABC292C] Four Variables

2023-04-25 2 min read # AtCoder # 题解

题目

  • 有一个正整数 nnn。
  • 求有多少个正整数组 (a,b,c,d)(a,b,c,d)(a,b,c,d) 使得 a⋅b+c⋅d=na\cdot b+c\cdot d=na⋅b+c⋅d=n。
  • 1≤n≤2×1051\le n\le 2\times10^51≤n≤2×105。

暴力做法

先枚举 a⋅ba\cdot ba⋅b，再求出 c⋅dc\cdot dc⋅d，循环里再枚举 a,ba,ba,b 中的较小数和 c,dc,dc,d 中的较小数，最后用加法原理和乘法原理求方案数。

代码实现

#include<bits/stdc++.h>
using namespace std;
int n,sum1,sum2,ans1,ans2;long long ans;
int main(){
	scanf("%d",&n);
	for(int i=1;i<n;i++){
		sum1=i,sum2=n-i;
		ans1=0,ans2=0;
		for(int j=1;j<=sqrt(sum1);j++) if(sum1%j==0){
			if(j*j==sum1) ans1++;
			else ans1+=2;
		}
		for(int j=1;j<=sqrt(sum2);j++) if(sum2%j==0){
			if(j*j==sum2) ans2++;
			else ans2+=2;
		}
		ans+=ans1*ans2;
	}
	printf("%lld",ans);
	return 0;
}
