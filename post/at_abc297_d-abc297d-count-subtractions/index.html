AT_abc297_d [ABC297D] Count Subtractions

2023-04-25 2 min read # AtCoder # 题解

题目

  • 有 222 个正整数 xxx 和 yyy。
  • 需要执行若干次操作直到 x=yx=yx=y。
  • 对于每次操作，分 222 种情况：
    • 若 x>yx>yx>y，x←x−yx\gets x-yx←x−y。
    • 若 x<yx<yx<y，y←y−xy\gets y-xy←y−x。
  • 求需要执行多少次操作。
  • 1≤x,y≤10181\le x,y\le10^{18}1≤x,y≤1018。

模拟+优化做法

设 ansansans 为执行操作的次数。当 x>yx>yx>y 时，不用一个一个减，只需执行 ans←ans+xyans\gets ans+\dfrac{x}{y}ans←ans+yx​，再执行 x←x mod yx\gets x\bmod yx←xmody 即可，这样做的前提是始终需要保持 x>yx>yx>y，直到 x=yx=yx=y 时退出。输出的答案需要特别注意，由于最后一次执行上述过程时，本来需要使 x=yx=yx=y，但因为 x mod y=0x\bmod y=0xmody=0，最后 xxx 会多减 111 个 yyy，所以答案并不是 ansansans，而是 ans−1ans-1ans−1。

代码实现

#include<bits/stdc++.h>
using namespace std;
long long ans;
void dfs(long long x,long long y){
	if(x<y) swap(x,y);
	if(y==0) return ;
	ans+=x/y,dfs(x%y,y);
}
int main(){
	long long x,y;scanf("%lld%lld",&x,&y);
	dfs(x,y);printf("%lld",ans-1);
	return 0;
}
