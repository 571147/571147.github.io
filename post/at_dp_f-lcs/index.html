题目

  • 给定两个字符序列 sss 和 ttt。
  • 求 sss 和 ttt 的最长公共子序列。
  • 1≤∣s∣,∣t∣≤30001\le|s|,|t|\le30001≤∣s∣,∣t∣≤3000。

子序列的定义

一个给定序列的子序列，是在该序列中删去若干元素后得到的序列。一个长度为 nnn 的序列，有 2n2^n2n 个子序列。

最长公共子序列

给定两个序列 XXX 和 YYY，当另一序列 ZZZ 既是 XXX 的子序列，又是 YYY 的子序列时，称 ZZZ 时序列 XXX 和 YYY 的公共子序列。最长公共子序列时长度最长的公共子序列。

但这还没有结束，还要把 sss 和 ttt 的最长公共子序列求出来。

之前求长度是从前往后推，那现在求序列就得从后往前推。

代码实现

#include<bits/stdc++.h>
using namespace std;
int dp[3010][3010];
char s[3010],t[3010],ans[3010];
int main(){
	scanf("%s%s",s+1,t+1);
	int n=strlen(s+1),m=strlen(t+1);
	for(int i=1;i<=n;i++)
		for(int j=1;j<=m;j++)
			if(s[i]==t[j]) dp[i][j]=dp[i-1][j-1]+1;
			else dp[i][j]=max(dp[i-1][j],dp[i][j-1]);
	int i=n,j=m;
	while(dp[i][j]>0){
		if(s[i]==t[j]) ans[dp[i][j]]=s[i],i--,j--;
		else if(dp[i-1][j]>dp[i][j-1]) i--;
		else j--;
	}
	for(int i=1;i<=dp[n][m];i++) printf("%c",ans[i]);
	return 0;
}
