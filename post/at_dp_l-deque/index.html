AT_dp_l Deque

2023-04-25 5 min read # AtCoder # 题解

题目

  • 有一个长度为 nnn 的双端队列。
  • 里面有数 a1,a2,a3,…,ana_1,a_2,a_3,\ldots,a_na1​,a2​,a3​,…,an​。
  • 双方轮流取数，每次只能取两端的数，得分为取的所有数的和。
  • 求在双方都采用最优策略时，先手的得分减去后手的得分。
  • 1≤n≤30001\le n\le 30001≤n≤3000。

区间 DP 做法

我们先设 dpl,rdp_{l,r}dpl,r​ 表示当先手先取，且只取完 [l,r][l,r][l,r] 范围内的数时，先手所能获得的最大得分(双方都采用最优策略)。

  • 假设先手先取 ala_lal​：

    • 这时后手可取 al+1a_{l+1}al+1​，剩下的数为 [l+2,r][l+2,r][l+2,r]，即先手在这轮之前能得到 dpl+2,rdp_{l+2,r}dpl+2,r​。
    • 这时后手也可取 ara_{r}ar​，剩下的数为 [l+1,r−1][l+1,r-1][l+1,r−1]，即先手在这轮之前能得到 dpl+1,r−1dp_{l+1,r-1}dpl+1,r−1​。
    • 后手肯定选择对先手不利的情况，即先手只能取到 al+min⁡(dpl+2,r,dpl+1,r−1)a_l+\min{(dp_{l+2,r},dp_{l+1,r-1})}al​+min(dpl+2,r​,dpl+1,r−1​)。

  • 假设先手先取 ara_rar​：

    • 这时后手可取 ar−1a_{r-1}ar−1​，剩下的数为 [l,r−2][l,r-2][l,r−2]，即先手在这轮之前能得到 dpl,r−2dp_{l,r-2}dpl,r−2​。
    • 这时后手也可取 ala_{l}al​，剩下的数为 [l+1,r−1][l+1,r-1][l+1,r−1]，即先手在这轮之前能得到 dpl+1,r−1dp_{l+1,r-1}dpl+1,r−1​。
    • 后手肯定选择对先手不利的情况，即先手只能取到 ar+min⁡(dpl,r−2,dpl+1,r−1)a_r+\min{(dp_{l,r-2},dp_{l+1,r-1})}ar​+min(dpl,r−2​,dpl+1,r−1​)。

先手肯定会选择得分更大的策略，即 max⁡(al+min⁡(dpl+2,r,dpl+1,r−1),ar+min⁡(dpl,r−2,dpl+1,r−1))\max{(a_l+\min{(dp_{l+2,r},dp_{l+1,r-1})},a_r+\min{(dp_{l,r-2},dp_{l+1,r-1})})}max(al​+min(dpl+2,r​,dpl+1,r−1​),ar​+min(dpl,r−2​,dpl+1,r−1​))。

初始值

因为上述做法肯定要使 l+2≤r,l≤r−2,l+1≤r−1l+2\le r,l\le r-2,l+1\le r-1l+2≤r,l≤r−2,l+1≤r−1，即 r−l+1≥3r-l+1≥3r−l+1≥3，所以 r−l+1≤2r-l+1\le 2r−l+1≤2 就得特殊处理。

  • 若 r−l+1=1r-l+1=1r−l+1=1，即 l=rl=rl=r，dpi,jdp_{i,j}dpi,j​ 无疑是 ala_lal​ (也可以写成 ara_rar​)。
  • 若 r−l+1=2r-l+1=2r−l+1=2，先手肯定会选择得分更大的策略，即 max⁡(al,ar)\max{(a_l,a_r)}max(al​,ar​)。

注意

千万记得开 long long。

代码实现

#include<bits/stdc++.h>
using namespace std;
int n,a[3010];long long ans,dp[3010][3010];
int main(){
    scanf("%d",&n);
    for(int i=1;i<=n;i++) scanf("%d",&a[i]),ans+=a[i];//求总数
    for(int i=1;i<=n;i++) dp[i][i]=a[i];//赋初始值
    for(int i=1;i<n;i++) dp[i][i+1]=max(a[i],a[i+1]);//赋初始值
    for(int len=3;len<=n;len++){//长度从 3 开始枚举
        for(int l=1;l<=n-len+1;l++){//枚举区间左端点
            int r=l+len-1;//求出区间右端点
            long long a1=a[l]+min(dp[l+2][r],dp[l+1][r-1]);//当先手取 a[l]
            long long a2=a[r]+min(dp[l][r-2],dp[l+1][r-1]);//当先手取 a[r]
            dp[l][r]=max(a1,a2);//两种方案取最优
        }
    }
    printf("%lld",dp[1][n]-(ans-dp[1][n]));//求差
    return 0;
}
