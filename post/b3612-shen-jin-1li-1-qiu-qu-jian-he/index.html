B3612 【深进1.例1】 求区间和

2023-04-25 2 min read # 洛谷 # 题解

题目

  • 有 nnn 个数 a1,a2,a3...ana_{1},a_{2},a_{3}...a_{n}a1​,a2​,a3​...an​。
  • 有 mmm 个区间 [li,ri][l_i,r_i][li​,ri​]。
  • 要求出每个区间的和。
  • 1≤n,m≤1051\le n,m\le10^51≤n,m≤105。

前缀和

我们设 sumi=∑k=1naksum_{i}=\sum\limits_{k=1}^n a_ksumi​=k=1∑n​ak​，那么 ∑k=lrak=∑k=1rak−∑k=1l−1ak=sumr−suml−1\sum\limits_{k=l}^r a_k=\sum\limits_{k=1}^r a_k-\sum\limits_{k=1}^{l-1} a_k=sum_{r}-sum_{l-1}k=l∑r​ak​=k=1∑r​ak​−k=1∑l−1​ak​=sumr​−suml−1​，只需在最开始把 sumsumsum 数组预处理好即可。

代码实现

#include<bits/stdc++.h>
using namespace std;
int n,m,l,r,a[100010],sum[100010];
int main(){
	scanf("%d",&n);
	for(int i=1;i<=n;i++) scanf("%d",&a[i]);
	for(int i=1;i<=n;i++) sum[i]=sum[i-1]+a[i];
	scanf("%d",&m);
	while(m--){
		scanf("%d%d",&l,&r);
		printf("%d\n",sum[r]-sum[l-1]);
	}
	return 0;
}
