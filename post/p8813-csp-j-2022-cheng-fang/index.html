题目

如果 aba^bab 的值不超过 109{10}^9109，则输出 aba^bab 的值，否则输出 -1。数据保证 1≤a,b≤1091 \le a, b \le {10}^91≤a,b≤109。

引子

这道题其实用暴力法也能做，因为 aaa 为正整数，除了 a=1a=1a=1 可能超时之外，其他的情况顶多乘 303030 次就超 10910^9109 了，所以只需要特判111的情况即可(这里就不贴代码了)，不过如果题目要求输出 abmod  109a^b\mod10^9abmod109 ，就通不过了，O(b)\mathcal O(b)O(b) 的复杂度显然太高，有没有更好的方案呢？

快速幂

快速幂可快多了，复杂度只有 O(log⁡2b)O(\log_2b)O(log2​b)，接下来我就来讲讲快速幂。

接下来我们考虑一个问题：

Q:Q:Q: 如何得到 a1,a2,a4,a8,a16......a^1,a^2,a^4,a^8,a^{16}......a1,a2,a4,a8,a16......?

A:A:A: 只需要在拆出 (b)2(b)_2(b)2​ 的每一位，并将得数累乘后，执行 a⇐a2a\Leftarrow a^2a⇐a2 即可。

因为要拆出 (b)2(b)_2(b)2​ 的每一位，所以复杂度为 O(log⁡2b)O(\log_2b)O(log2​b)。

注意

千万记得开 long long。

代码实现

#include<bits/stdc++.h>
using namespace std;
long long mypow(long long a,long long b){//快速幂
		long long ans=1;//得数
		while(b>0){
			if(a>1e9) return -1;
			if(b&1) ans*=a;//若 b 在二进制下这一位为 1；"b&1" 也可以写成 "b%2==1"
			if(ans>1e9) return -1;
			a*=a,b>>=1;"b>>=1" 也可以写成 "b/=2"
		}
		return ans;
}
int main(){
		long long a,b;//a 为底数，b 为指数
		scanf("%lld%lld",&a,&b);
		printf("%lld",mypow(a,b));
}
