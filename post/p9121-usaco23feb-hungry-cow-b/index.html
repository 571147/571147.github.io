P9121 [USACO23FEB] Hungry Cow B

2023-04-25 3 min read # 题解

题目

  • 有一位农场主约翰，他有一只饥饿的牛贝西。
  • 每一天晚上，如果农场里有至少 111 堆干草，那么贝西吃且仅吃 111 堆干草。
  • 有 nnn 个事件，第 iii 个事件表示第 did_idi​ 天早晨约翰搬运了 bib_ibi​ 堆干草到农场。
  • 求第 1∼t1\thicksim t1∼t 天贝西共吃掉干草堆的数量。
  • 1≤n≤1051\le n\le 10^{5}1≤n≤105，1≤t≤10141\le t\le10^{14}1≤t≤1014，1≤di≤10141\le d_i\le 10^{14}1≤di​≤1014，1≤bi≤1091\le b_i\le 10^{9}1≤bi​≤109。
  • 1≤d1<d2<⋯<dn≤t1\le d_1<d_2<\cdots<d_n\le t1≤d1​<d2​<⋯<dn​≤t。

模拟做法

设 sumsumsum 为目前农场里剩余干草堆的数量，ansansans 为目前贝西共吃掉干草堆的数量。

对于每个 iii，我们不妨设 pospospos 为第 di−1∼di−1d_{i-1}\thicksim d_{i}-1di−1​∼di​−1 天贝西共吃掉干草堆的数量，则有 pos=max⁡(sum,di−di−1)pos=\max(sum,d_i-d_{i-1})pos=max(sum,di​−di−1​)，并执行 sum←sum−possum\gets sum-possum←sum−pos 与 ans←ans+posans\gets ans+posans←ans+pos，最后不能忘了执行 sum←sum+bisum\gets sum+b_isum←sum+bi​。

千万要注意，第 dn∼td_n\thicksim tdn​∼t 天共吃掉干草堆的数量还没加上，所以最终要输出 ans+min⁡(sum,t−dn+1)ans+\min(sum,t-d_n+1)ans+min(sum,t−dn​+1)，而不是直接输出 ansansans。

代码实现

#include<bits/stdc++.h>
using namespace std;
long long n,t,sum,ans,pos,d[100010],b[100010];
int main(){
	scanf("%lld%lld",&n,&t);
	for(long long i=1;i<=n;i++){
		scanf("%lld%lld",&d[i],&b[i]);
		pos=min(sum,d[i]-d[i-1]);
		sum-=pos,ans+=pos,sum+=b[i];
	}
	printf("%lld",ans+min(sum,t-d[n]+1));
	return 0;
}
